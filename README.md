# kabum-chk
Checker de contas Kabum em python!

## Como usar
Edite o arquivo .txt ``lista.txt`` com o formato de combo ``(email|senha)``

**Exemplo:**
- ``kplegal@gmail.com|senha123``

## Requirements
``pip install requests``
